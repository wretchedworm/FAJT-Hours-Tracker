# FAJT Hours

A mobile-first work-hours tracker for two monthly pay cycles: the 1st–15th and the 16th–last day.

## Open locally

Double-click `index.html`. The app saves its records in that browser on that device.

For installable/offline mode, run:

```sh
npm start
```

Then open `http://localhost:4173`.

## GitHub and syncing

This directory is a local Git repository, but it is not connected to GitHub yet. Cloud login, cross-device syncing, hosting, and deployment are not part of the current local version.
